# React Dark Mode Toggle

This project is a simple React application that allows users to switch between light and dark themes. The user's theme preference is saved in the local storage and persists across sessions. The default theme is based on the system's preferences if no saved setting is found.

## 📝 Story: Implement Dark Mode Toggle

### Description:
As a user, I want to switch between light and dark themes in the app, so that I can choose the display mode that suits my environment or preference.

### Acceptance Criteria:
- A toggle button is visible in the top-right corner of the app.
- Clicking the toggle switches the UI between dark and light themes.
- User’s choice is saved in localStorage and persists across sessions.
- Default theme is based on system preferences (if no saved setting).

## How to Use
Click the toggle button in the top-right corner of the app to switch between light and dark themes. Your choice will be saved and applied in your future sessions.

## Dependencies
This project uses the following dependencies:
- react: ^18.2.0
- react-dom: ^18.2.0
- react-scripts: 5.0.1

## Development Setup
1. Clone the repository.
2. Install the dependencies using `npm install`.
3. Start the development server using `npm start`.
